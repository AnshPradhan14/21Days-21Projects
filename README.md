## Overview

Welcome to my **21-Day AI Code Marathon**! 

The goal of this repository is to build 21 distinct projects in 21 days, covering the full spectrum of Artificial Intelligence. The journey starts with **Classical Machine Learning** to build a strong foundation, moves into **Deep Learning & Computer Vision**, and culminates with cutting-edge **Generative AI and Agentic Workflows**.

Each folder in this repository corresponds to a specific day and contains the code, datasets, and documentation for that project.

---

## Tech Stack

**Languages:** Python  
**Libraries & Frameworks:**
* **ML:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn, XGBoost
* **Deep Learning:** TensorFlow/Keras, PyTorch, OpenCV
* **GenAI & LLMs:** LangChain, Hugging Face, OpenAI API, Groq, Streamlit, ChromaDB
* **Agents & Automation:** CrewAI/Phidata, Selenium, n8n
